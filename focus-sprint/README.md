# Focus Sprint

A small browser-based Pomodoro-style focus timer with task tracking.

## Features

- Focus / Short Break / Long Break modes with configurable durations
- Circular progress ring showing time remaining
- Task list — add tasks, select one to focus on, track completed pomodoros per task
- Daily session streak counter (persisted in `localStorage`)
- Chime and celebration screen when a sprint or break ends

## Running it

No build step or dependencies — just open `index.html` in a browser.

## Files

- `index.html` — markup
- `style.css` — styling
- `app.js` — timer logic, task management, and persistence
